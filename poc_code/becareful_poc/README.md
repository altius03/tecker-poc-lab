# Be:Careful PoC

Be:Careful 약 복용 안내 PoC 프로젝트 위치입니다.

현재 공유 레포에는 제품 문서와 Codex 작업지시서가 포함되어 있고, 실행 가능한 Python 구현 코드는 아직 원본 작업 폴더에 생성되지 않았습니다.

## 관련 문서

- `../../docs/02_BeCareful.md`
- `../../docs/codex_poc_instructions/02_BeCareful_Codex_PoC_작업지시서.md`

## 구현 결과가 들어오면 포함할 항목

- `pyproject.toml`
- `.env.example`
- `samples/`
- `src/`
- `tests/`
- `tools/`

Be:Careful은 더미 OCR과 mock DUR 기준이므로 실제 API 키 없이 실행 가능해야 합니다.
