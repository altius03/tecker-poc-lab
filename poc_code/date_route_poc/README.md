# 데이트 동선 탐색 PoC

데이트 동선 탐색 PoC 프로젝트 위치입니다.

현재 공유 레포에는 제품 문서와 Codex 작업지시서가 포함되어 있고, 실행 가능한 Python 구현 코드는 아직 원본 작업 폴더에 생성되지 않았습니다.

## 관련 문서

- `../../docs/03_데이트_동선.md`
- `../../docs/codex_poc_instructions/03_데이트_동선_Codex_PoC_작업지시서.md`

## 구현 결과가 들어오면 포함할 항목

- `pyproject.toml`
- `.env.example`
- `samples/`
- `src/`
- `tests/`
- `tools/`

실제 Kakao Local API 검증 시에는 `.env.example`을 `.env`로 복사하고 `KAKAO_REST_API_KEY`를 로컬에서만 설정합니다.
